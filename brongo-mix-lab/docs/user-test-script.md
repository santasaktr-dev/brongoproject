# BRONGO Mix Lab — User Test Script

สคริปต์รันเซสชันทดสอบกับผู้ปกครอง 5–8 คน (อ้างอิง `../../docs/user-testing-plan.md`) 30–40 นาที/คน บนมือถือของผู้เข้าร่วมเมื่อทำได้

## บทเปิด (moderator)

> "วันนี้เรากำลังทดสอบเว็บไซต์ ไม่ได้ทดสอบคุณ บางจุดอาจยังไม่สมบูรณ์ ขอให้ลองใช้ตามธรรมชาติ หากสงสัยให้พูดออกมาได้ แต่ระหว่างภารกิจฉันอาจยังไม่ตอบทันที เพื่อดูว่าเว็บไซต์อธิบายตัวเองได้หรือไม่ กิจกรรมนี้ไม่ใช่คำแนะนำทางการแพทย์ และเราจะไม่ถามข้อมูลสุขภาพของคุณหรือบุตร"

**ไม่เก็บ:** ชื่อเด็ก, โรค, ยาที่ใช้อยู่, ประวัติการรักษา

## Tasks

| # | Prompt | สังเกต | Pass |
|---|---|---|---|
| 1 | "ดูหน้าแรก แล้วบอกว่าเว็บนี้ให้ทำอะไร" | เข้าใจว่าเป็นเกมเลือกส่วนผสม, เห็น CTA ใน 5 วิ | เข้าใจว่าเกี่ยวกับ BRONGO |
| 2 | "เลือกส่วนผสมที่สนใจอย่างน้อย 3 รายการ แล้วดูผล" | เจอครบ 5 หมวด, เข้าใจ selected state, เข้าใจ Vitamin B6/ไม่ใส่ | ทำถึงหน้า Result |
| 3 | "คะแนนนี้หมายความว่าอะไร" | — | เข้าใจว่าเป็นสัดส่วนที่ตรงกับ BRONGO ไม่ใช่ความปลอดภัย/ประสิทธิภาพ; 100% ไม่ได้แปลว่าเลือกครบสูตร |
| 4 | "หาว่าส่วนผสมไหนมีใน BRONGO และอ่านเรื่องน่ารู้ 1 รายการ" | แยก matched/unmatched โดยไม่พึ่งสีอย่างเดียว, เจอ fun fact | เข้าใจว่าบางรายการไม่มี fun fact |
| 5 | "กลับไปแก้สูตรเดิม เปลี่ยน 1 รายการ แล้วดูผลอีกครั้ง" | หา CTA แก้สูตรเจอ, สูตรเดิมยังอยู่ | เข้าใจว่าคะแนนเปลี่ยนเพราะ matches/denominator |
| 6 | "หาข้อมูลว่าอาการแบบใดต้องหยุดยาและพบแพทย์ทันที" | เข้า BRONGO Detail, ใช้ accordion, เจอใน 30 วิ | แยกข้อมูลเกมออกจากข้อมูลยาได้ |

## คำถามหลังทดสอบ

1. ส่วนไหนสนุกที่สุด? 2. ส่วนไหนสับสน? 3. คะแนนหมายถึงอะไร? 4. เว็บกำลังแนะนำให้เลือกยาหรือไม่? 5. ข้อมูลส่วนผสมอ่านง่าย 1–5? 6. ภาพ/animation เหมาะไหม? 7. มีข้อความโฆษณาเกินจริงไหม? 8. อยากเล่นซ้ำ/แนะนำต่อไหม?

## Metrics เป้าหมาย

Landing→Start ≥90% · Start→Result ≥85% · เข้าใจคะแนน ≥80% · ไม่ตีความว่าวินิจฉัย/เลือกยา ≥90% · เจอ safety ใน 30 วิ ≥80% · accidental tap ≤2/session · usability ≥4/5

## Severity & release rule

Critical (flow พัง / ข้อมูลยาผิด / เข้าใจว่าเป็นคำแนะนำแพทย์) ต้อง = 0 · High แก้ก่อนรอบถัดไป · ทำซ้ำ 3–5 คนหลังแก้ Critical/High
