# BRONGO Mix Lab — Mascot Reactions

## บทบาทของ Mascot

Mascot เป็นผู้ช่วยประจำ Mix Lab ทำหน้าที่ชวนเล่น แสดงปฏิกิริยา และพาผู้ใช้ไปอ่านข้อมูล ไม่ทำหน้าที่เป็นแพทย์หรือผู้รับรองว่าสูตรของผู้ใช้เหมาะกับการรักษา

## Voice Principles

- อบอุ่น สนุก และไม่ตำหนิ
- ใช้ประโยคสั้น อ่านร่วมกับเด็กได้
- หลีกเลี่ยงคำว่า “ยาที่ใช่สำหรับคุณ”, “ปลอดภัยกว่า”, “ดีที่สุด” หรือ “ควรเลือก” ในหน้าผล
- คะแนนหมายถึงความตรงของรายการที่เลือกกับ BRONGO เท่านั้น
- ทุก result ต้องมีคำอธิบายคะแนนหนึ่งบรรทัด

## Contextual Reactions

### Landing

- Idle: หายใจและกะพริบตาช้า ๆ
- CTA hover/focus: mascot ชี้ไปที่ปุ่ม “เริ่มผสมยา”
- ข้อความ: `พร้อมเข้าห้องทดลองหรือยัง?`

### Ingredient Selection

- เลือกรายการ: พยักหน้าหรือชูช้อน ระยะไม่เกิน 350 ms
- ยกเลิก: กลับสู่ idle โดยไม่แสดงสีหน้าเชิงลบ
- เลือกครบ 3 รายการ: `สูตรเริ่มเป็นรูปเป็นร่างแล้ว!`
- เลือกครบ 6 รายการขึ้นไป: `ส่วนผสมเยอะขึ้นแล้ว พร้อมดูผลเมื่อไรก็กดได้เลย`

### Empty Submission

- CTA disabled จึงไม่ควรเกิด submission
- หาก state ผิดพลาด ให้แสดง mascot ถือหลอดทดลองเปล่า
- ข้อความ: `เลือกอย่างน้อย 1 ส่วนผสมก่อนนะ`

### Product Detail

- ใช้ mascot เฉพาะ header และ CTA ท้ายหน้า
- ไม่มีการกระโดดหรือ confetti ระหว่างอ่านข้อมูลยา
- ข้อความท้ายหน้า: `อ่านข้อมูลแล้ว อยากลองผสมสูตรใหม่ไหม?`

## Motion Specification

- Idle loop: 3–5 วินาที ไม่ขยับตลอดเวลา
- Selection response: 200–350 ms
- Mixing anticipation: 500–700 ms
- Result entrance: 400–600 ms
- Perfect confetti: ไม่เกิน 1.2 วินาที และไม่เล่นซ้ำเอง
- Reduced motion: แทนการกระโดด/เขย่าด้วย opacity transition 120–180 ms
- Animation ต้องไม่ส่งเสียงอัตโนมัติ

## Approved Base Assets

- `Logo.png`: mascot สีเหลือง ตาเปิด — base สำหรับ `idle`, `pointing`, `curious` และ `empty`
- `Logo2.png`: mascot สีส้ม ตายิ้ม — base สำหรับ `mixing`, `happy`, `almost-perfect` และ `perfect`
- ทั้งสองไฟล์มี alpha channel และใช้ได้โดยตรงบนพื้นหลังเว็บไซต์
- สร้างความแตกต่างของ state ด้วยตำแหน่ง การหมุนเล็กน้อย การ scale, props และ particles รอบตัว ห้ามแก้สีหรือวาดทับใบหน้าต้นฉบับ

## Runtime State Mapping

| State | Base asset | Runtime treatment |
|---|---|---|
| `idle` | `Logo.png` | float เบา ๆ |
| `pointing` | `Logo.png` | วางข้าง CTA พร้อมลูกศร/เส้นนำสายตา |
| `mixing` | `Logo2.png` | rotate สลับเล็กน้อยและเพิ่ม vessel effect |
| `retry` | `Logo.png` | ลด bounce และเพิ่มฟองหนึ่งจุด |
| `curious` | `Logo.png` | tilt เล็กน้อยและเพิ่มแว่นขยายเป็น prop แยก |
| `happy` | `Logo2.png` | bounce หนึ่งครั้ง |
| `almost-perfect` | `Logo2.png` | เพิ่มดาวสองดวง |
| `perfect` | `Logo2.png` | เพิ่ม confetti สั้น ๆ |
| `empty` | `Logo.png` | วางคู่กับหลอดทดลองเปล่า |
